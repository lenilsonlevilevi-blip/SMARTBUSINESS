SmartBusiness - ERP para Pequenos Negócios


⚠️ Projeto em desenvolvimento ativo. Este sistema está sendo construído aos poucos, como parte do meu processo de aprendizado e evolução como desenvolvedor. Funcionalidades podem mudar, quebrar ou ser reescritas com frequência.
Sistema de gestão empresarial (ERP) voltado para pequenos negócios, com o objetivo de centralizar o controle de clientes, produtos, financeiro e operações do dia a dia em um só lugar.

🚧 Status do projeto

Este projeto está na fase inicial de construção. A estrutura básica (dashboard, clientes, produtos e financeiro) já está funcionando, mas ainda há muita coisa a ser implementada, testada e refinada.

Funcionalidades

✅ Já implementado
Dashboard inicial
Tela de clientes
Tela de produtos
Tela financeira


🔜 Em planejamento
 Relatório de vendas
 Autenticação de usuários (login/logout)
 Controle de estoque
 Cadastro completo de clientes e produtos (CRUD)
 Emissão de relatórios em PDF
 Testes automatizados


Tecnologias utilizadas


Python — linguagem principal do backend
Flask — micro-framework web
HTML5 / CSS3 — estrutura e estilização das páginas
Jinja2 — templates dinâmicos (nativo do Flask)
